Passwortspeicherung in Chromium

Das Speichern von Passwörtern im Gnome-Schlüsselbund wurde behoben, aber damit es funktioniert, müssen Sie Ihre alte Datei ~/snap/chromium/common/chromium/Default/Login Data umbenennen, damit eine neue, funktionierende Datei sie ersetzen kann.

Wir empfehlen Ihnen, sie in Login Data.old umzubenennen. Dafür können Sie Ihren Dateimanager benutzen oder ein Terminal öffnen und das Folgende eingeben: 

cd ~/snap/chromium/common/chromium/Default/
mv "Login Data" "Login Data.old"

Auf diese Weise können Sie die Aktion jederzeit rückgängig machen, indem Sie die Datei wieder in Login Data umbenennen, falls etwas fehlschlägt. Bitte reichen Sie in diesem Fall einen Fehlerbericht in Launchpad ein.

Um sorgfältig vorsichtig zu sein, gehen Sie zur Einstellungen > Passwortmanager (about://settings/passwords), klicken Sie das drei-Punkte Symbol neben Gespeicherte Passwörter und dann Passwörter exportieren in eine CSV Datei. Diese können Sie später importieren, um Ihre Passwörter wiederherzustellen.

Um diese Meldung zu deaktivieren, erstellen Sie die Datei ~/snap/chromium/common/chromium/Default/Login Data.old.
